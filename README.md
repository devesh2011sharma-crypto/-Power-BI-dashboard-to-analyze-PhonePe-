# -Power-BI-dashboard-to-analyze-PhonePe-
🍛 Developed an interactive Power BI dashboard to analyze PhonePe 💳📊
🍛 Developed an interactive Power BI dashboard to analyze PhonePe 💳📊transaction data and provide actionable insights into payment behavior, transaction performance, and user demographics. 


The dashboard tracks key metrics such as total transactions, transaction value, successful payment rate, active users, service-wise revenue contribution, age-segment distribution, and monthly transaction trends. 

Implemented dynamic filters, KPIs, drill-down analysis, and data visualizations to monitor payment status, identify business patterns, and support data-driven decision-making.
